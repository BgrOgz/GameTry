# DriftRacerMobile

A 2D top-down mobile car drifting game built with Unity.

## Features

- Dual control schemes (Button controls & Steering wheel)
- Multiple cars to unlock
- Multiple tracks/levels
- Upgrade system
- Drift-based scoring with combos
- Cross-platform (iOS & Android)

## Development

- **Unity Version**: 2022.3 LTS
- **Platform**: iOS & Android
- **Development Timeline**: 12 weeks

## Project Structure

```
Assets/_Project/
├── Scenes/          # Game scenes
├── Scripts/         # C# scripts
├── Prefabs/         # Reusable game objects
├── ScriptableObjects/  # Game data
├── Sprites/         # 2D graphics
└── Audio/           # Music and SFX
```
